# Metric to Imperial Unit Converter

[A live demo of the app is available here.](https://unitconverterdhk.netlify.app/)

This is a simple web application to convert between metric and imperial units for length, volume, and mass. Enter a number and click **Convert** to see the following conversions:

- **Meters** ↔ **Feet**
- **Liters** ↔ **Gallons**
- **Kilograms** ↔ **Pounds**

Conversion results are displayed instantly, showing conversions both ways for each measurement.  
The tool is made with plain HTML, CSS, and JavaScript **no external dependencies required**.
